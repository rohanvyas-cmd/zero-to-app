Chapter 05

Build your first app.

Time to make it real. In the next ~30 minutes you'll create a React app, add a feature with Claude Code, push it to GitHub, and deploy it to a live URL. Follow along in your terminal.

1

Scaffold a Vite React app

From any folder you want to keep projects in.

Terminal 
cd ~/Desktop
npm create vite@latest zero-to-app -- --template react
cd zero-to-app
npm install
You should see a zero-to-app folder on your Desktop.
2

Open it in VS Code

From the same terminal.

Terminal 
code .
If code isn't found, open VS Code → ⌘⇧P → "Shell Command: Install 'code' command in PATH".
3

Run the dev server

Open a new terminal inside VS Code (⌃ + `) and start it.

Terminal 
npm run dev
Open http://localhost:5173. You should see the default Vite + React page.
4

Add a feature with Claude Code

In a second terminal, fire up Claude and ask for something fun.

Terminal 
claude
Try this prompt
Replace the default landing page with a simple "mood journal": a text area, a save button, and a list of saved moods. Store them in localStorage so they persist. Make it look clean and modern.
Your browser should auto-refresh with your new mood journal.
5

Push to GitHub

One command, thanks to the gh CLI you installed in Chapter 02.

Terminal 
git init
git add .
git commit -m "first ship"
gh repo create zero-to-app --public --source=. --push
Prefer the manual way? Create an empty repo at github.com/new and run git remote add origin <url> then git push -u origin main.
6

Deploy to Vercel

  1. Go to vercel.com/new.
  2. Sign in with GitHub, click "Import" on your new repo.
  3. Accept all defaults (Vercel auto-detects Vite).
  4. Click Deploy.
You just shipped your first app. Vercel gave you a live URL like zero-to-app.vercel.app. Text it to someone right now.
7

Ship again (the real superpower)

Tweak the app with Claude, then push. Vercel redeploys automatically.

Terminal 
git add .
git commit -m "added dark mode"
git push
Watch Vercel's dashboard — a new deploy starts within seconds. This is your workflow forever now.
