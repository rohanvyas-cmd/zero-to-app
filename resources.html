Command cheatsheet

brew install <pkg> Install anything on macOS
npm install Install a project's dependencies
npm run dev Start a local dev server
npm create vite@latest Scaffold a new frontend app
claude Open Claude Code in the current folder
git status See what files you've changed
git add . && git commit -m "…" Save a snapshot of your work
git push Upload your commits (triggers Vercel deploys)
gh repo create … --push Create a GitHub repo and push in one step
git restore . Undo all uncommitted changes (careful!)
code . Open current folder in VS Code
⌃ + c Stop whatever is running in the terminal

When things break

Terminal: command not found
The tool isn't installed — or your shell hasn't reloaded. Close and reopen the terminal, or run source ~/.zshrc.
npm error: EACCES / permission denied
Never use sudo with npm. Use nvm to manage Node versions instead of the system one.
Git: "Please tell me who you are"
You skipped a step in Chapter 02. Run git config --global user.name "…" and git config --global user.email "…".
git push asks for a password
GitHub killed password auth for git. Run gh auth login once and it'll wire up SSH keys for you.
"Port 5173 is already in use"
Another dev server is still running. Close that terminal tab or run npm run dev -- --port 3000.
Vercel build fails
Open the build log. 9 times out of 10 it's a missing env var or a typo in package.json. Copy the error into Claude and ask for a fix.
Claude Code made changes I don't understand
Ask: "walk me through what you just changed and why." Then git diff to see it line by line. If you hate it, git restore . to undo.
